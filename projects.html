Artificial Neural Network

The ANN has two layers, an input layer, and an output layer, and is trained using the backpropagation algorithm with stochastic gradient descent optimizer. The project uses a dataset with 7000 samples and 10 features each, and the performance of the model is evaluated based on the accuracy on the test dataset.

This project implements an Artificial Neural Network (ANN) for multiclass classification in Python using Numpy, Pandas, and Matplotlib.

Discord Bot with NLP

Harumi is a versatile Discord bot that keeps servers entertained and engaged. Currently, the bot is under development with plans to introduce a fully fledged game system, more responsive features for Discord servers, and moderator type features.
